# Инструкция по настройке SMS авторизации через SMS Aero

## Обзор

SMS Aero - российский сервис для отправки SMS сообщений. Эта инструкция поможет вам настроить SMS авторизацию через SMS Aero для вашего приложения.

## Шаг 1: Регистрация в SMS Aero

1. Перейдите на сайт [smsaero.ru](https://smsaero.ru/)
2. Нажмите кнопку **"Регистрация"** в правом верхнем углу
3. Заполните форму регистрации:
   - Email (будет использоваться для API)
   - Номер телефона
   - Пароль
4. Подтвердите регистрацию, перейдя по ссылке из письма на вашем email

## Шаг 2: Пополнение баланса

1. Войдите в личный кабинет SMS Aero
2. Перейдите в раздел **"Пополнение"** или **"Баланс"**
3. Пополните баланс минимум на **200₽** (минимальная сумма пополнения)
4. Стоимость одной SMS: от **1.8₽**

## Шаг 3: Настройка подписи отправителя

1. В личном кабинете перейдите в раздел **"Настройки"** → **"Подписи"**
2. Нажмите **"Добавить подпись"**
3. Введите название подписи (например, **"RevMarket"**)
4. Подождите подтверждения подписи администрацией SMS Aero (обычно 1-2 рабочих дня)
5. После подтверждения подпись будет доступна для использования

⚠️ **Важно:** Подпись должна быть подтверждена перед использованием в API!

## Шаг 4: Получение API ключа

1. В личном кабинете перейдите в раздел **"Настройки"** → **"API и SMPP"**
2. Нажмите кнопку **"Сгенерировать"** для создания нового API ключа
3. **Скопируйте и сохраните** сгенерированный API ключ (он показывается только один раз!)

📝 **Примечание:** 
- Email для API - это ваш email, который вы использовали при регистрации
- Пароль для API - это сгенерированный API ключ (НЕ ваш пароль от аккаунта!)

## Шаг 5: Настройка переменных окружения

### Для локальной разработки

Создайте файл `.env.local` в папке `sms-webhook/`:

```env
# SMS Provider Configuration
SMS_PROVIDER=smsaero
SMS_API_KEY=ваш_api_ключ_из_sms_aero
SMS_AERO_EMAIL=ваш_email@smsaero.ru
SMS_SENDER=RevMarket

# Supabase SMS Webhook Configuration
SUPABASE_SMS_SECRET=ваш_секрет_из_supabase
```

### Для Vercel (продакшен)

1. Перейдите в Vercel Dashboard → Ваш проект → Settings → Environment Variables
2. Добавьте следующие переменные:

| Переменная | Значение | Описание |
|------------|----------|----------|
| `SMS_PROVIDER` | `smsaero` | Провайдер SMS |
| `SMS_API_KEY` | Ваш API ключ | API ключ из SMS Aero |
| `SMS_AERO_EMAIL` | Ваш email | Email, использованный при регистрации |
| `SMS_SENDER` | `RevMarket` | Подпись отправителя (должна быть подтверждена) |
| `SUPABASE_SMS_SECRET` | Секрет из Supabase | Секрет для webhook (см. шаг 6) |

## Шаг 6: Настройка Supabase

1. Зайдите в [Supabase Dashboard](https://app.supabase.com/)
2. Выберите ваш проект
3. Перейдите в **Authentication** → **Settings** → **Phone Auth**
4. Включите следующие опции:
   - ✅ **Enable phone signup**: ON
   - ✅ **Enable phone login**: ON
   - ✅ **Enable Send SMS hook**: ON
5. Настройте SMS Hook:
   - **Hook type**: HTTPS
   - **URL**: `https://your-vercel-app.vercel.app/api/send-sms`
     (замените `your-vercel-app` на ваш домен Vercel)
   - **Secret**: Нажмите **"Generate secret"** или введите свой секрет
6. **Скопируйте сгенерированный секрет** и добавьте его в Vercel как `SUPABASE_SMS_SECRET`

## Шаг 7: Деплой webhook на Vercel

1. Убедитесь, что вы находитесь в папке `sms-webhook/`
2. Установите зависимости (если еще не установлены):
   ```bash
   npm install
   ```
3. Деплой на Vercel:
   ```bash
   vercel --prod
   ```
   Или используйте GitHub интеграцию Vercel для автоматического деплоя

## Шаг 8: Тестирование

1. Убедитесь, что все переменные окружения настроены
2. Проверьте, что подпись отправителя подтверждена в SMS Aero
3. Попробуйте авторизоваться в вашем приложении, введя номер телефона
4. Проверьте, что SMS пришло на указанный номер
5. Проверьте логи в Vercel Dashboard → Functions → `/api/send-sms` для отладки

## Проверка работы

### Успешная отправка SMS

Если все настроено правильно, вы должны увидеть в логах Vercel:
```
SMS sent via smsaero: { success: true, data: {...} }
```

### Возможные ошибки

1. **"SMS_AERO_EMAIL is not set"**
   - Проверьте, что переменная `SMS_AERO_EMAIL` установлена в Vercel

2. **"SMS_API_KEY is not set"**
   - Проверьте, что переменная `SMS_API_KEY` установлена в Vercel

3. **"SMS Aero error: Invalid signature"**
   - Подпись отправителя не подтверждена или указана неверно
   - Проверьте статус подписи в личном кабинете SMS Aero

4. **"SMS Aero error: Insufficient funds"**
   - Недостаточно средств на балансе
   - Пополните баланс в личном кабинете SMS Aero

5. **"Unauthorized"**
   - Неверный секрет от Supabase
   - Проверьте, что `SUPABASE_SMS_SECRET` в Vercel совпадает с секретом в Supabase

## Поддержка

Если у вас возникли проблемы:

1. Проверьте логи в Vercel Dashboard
2. Проверьте баланс в личном кабинете SMS Aero
3. Убедитесь, что подпись отправителя подтверждена
4. Обратитесь в поддержку SMS Aero: support@smsaero.ru

## Альтернативные провайдеры

Если SMS Aero не подходит, вы можете использовать:

- **Exolve** - от 1.5₽ за SMS (минимальное пополнение 100₽)
- **eSputnik** - от 2₽ за SMS (минимальное пополнение 500₽)

Для переключения провайдера измените переменную `SMS_PROVIDER` в Vercel.
